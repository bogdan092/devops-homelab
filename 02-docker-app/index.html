🏗️ 1. Infrastructure (Terraform)

Our foundation. Terraform talks to the KVM hypervisor on the HP Z4 to provision the virtual hardware.

  • main.tf: Defines the RAM, CPUs, and Network.
  • cloud_init.cfg: The boot script that installs Docker automatically on first launch.

🐳 2. Containerization (Docker)

We packaged our Nginx web server so it runs perfectly anywhere, avoiding the "it works on my machine" problem.

  • Dockerfile: The recipe to build our custom image.
  • docker-compose.yml: Our stepping stone before moving to full Kubernetes.

☸️ 3. Orchestration (Kubernetes)

The Fleet Admiral. K3s groups our two VMs into a single supercomputer for High Availability.

  • ConfigMap: Stores this exact HTML file in the cluster's memory.
  • Deployment: Ensures 2 copies of Nginx are always running.
  • Service: The LoadBalancer exposing Port 8080 to your browser.
