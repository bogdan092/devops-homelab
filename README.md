#Local DevOps HomeLab

This contains IaC and config files

##Tech Stack

*HostOS and Hypervisor: Ubuntu 24.04 and KVM
*IaC: Terraform
*Containerization: Docker
*Orchestration: Kubernetes and Ansible

## Project Structure

*01-terraform: Provisioning of linux VMs using libvirt provider
*02-docker: Dockerfiles and docker-comose config for custom applications
*03-kubernetes: YAML masnifests (deploys,services,ingress) for cluster workloads

*(This project is actively being built)
